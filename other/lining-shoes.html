标题：AI挑鞋-李宁休闲鞋
描述：本文介绍借助AI挑选李宁休闲鞋的方法，用Qwen3-Max分析性能推荐月白3、天迹2.0等鞋款，用Gemini3-flash选配色，最终选购月白3大地棕底款，并验证轻云科技缓震表现。
发布日期：2026/01/13
标签：
选鞋李宁Qwen3Gemini3

背景 ​

  • 不同型号功能的鞋，颜色款式也有很多，自己挑选难以选择
  • 借助AI帮助分析筛选，给出最合理的选择
  • 不同的模型能力不同，需要按需选择

型号选择 - Qwen3-Max ​

  • Qwen3-Max 作为一个大语言模型，语言能力出色，用来分析性能、功能，可以给出合理的选择

  • 但言语描述颜色是困难的，所以不要让它来选颜色搭配，它就是个瞎子！

  • 告诉 AI 你的身高体重（大体重，身高标准）、兴趣爱好（运动）、穿衣风格（平时穿的上衣裤子颜色风格）、穿鞋的场合（日常休闲、通勤、旅游，非重度运动），让它在指定品牌（李宁）中进行选择。

结论：兼顾旅游+通勤+休闲 ​

注意：AI给的编号，大概率是错的，原因一部分是在于网页上李宁款号的混乱

你需要的是：有一定缓震、鞋底不过厚、支撑稳定、不加绒、冷色调、42码、非跑鞋。推荐以下两款：

1. 李宁 月白3（型号：ARBL059） ​

  • 中底：轻云科技（比云科技更轻，缓震适中）
  • 鞋底厚度：约28mm，落差6mm，比跑鞋贴地，比板鞋舒适
  • 鞋面：致密工程网布 + 合成革，不闷脚、不加绒
  • 配色：有“砂砾灰/茗茶绿”、“标准白/黑”等冷调
  • 适用场景：通勤、半日旅游、日常休闲
  • 优势：李宁休闲线中缓震与稳定性平衡得最好的之一

2. 李宁 天迹2.0（型号：ABAR073） ​

  • 中底：李宁云科技，缓震更明显
  • 鞋底：全掌橡胶，耐磨防滑
  • 缺点：中底略厚（约30mm），重心稍高，不适合高强度训练，但适合走路
  • 配色：有“南极灰”、“云雾白”等你偏好的色调

✅ 这两款都明确标注了缓震科技，且在电商页面强调“日常通勤”“城市出行”“轻运动”等用途，功能定位更贴近你的实际需求。

对比总结 ​

鞋款缓震支撑适合走路？适合你？
月白3 (ARBL059)✅ 轻云科技✅ 中等✅ 适合半日游/通勤✅ 推荐
天迹2.0 (ABAR073)✅ 云科技✅ 中等✅ 适合日常行走✅ 可选

颜色选择 - Gemini3-flash ​

  • 作为一个多模态大模型，它可以分析图片，识别颜色，给出合理的选择
  • 作为一个海外模型，在训练数据上可能对中国数据不全面，让它分析一个本土品牌的功能、性能选择，恐怕不是好主意

结论 ​

  • 针对您 48岁、身高178cm、体重180斤 的身体条件，以及常穿 深蓝色/藏青色裤子和外套 的穿搭习惯，这款新选择的鞋型版型稳重，非常适合您的体型。

  • 在众多的配色中，为了避免您担心的“配色奇怪”或“纯白突兀”的问题，我为您选出最合适的 3 个配色：

🏆 首选推荐：【AGCV077-14 月白3】（云纹蓝红款） ​

这是为您量身定制的“高级感”方案。

  • 推荐理由： 鞋身侧面的深蓝色云纹与您的藏青色裤子和外套形成了完美的色彩呼应。鞋头和后跟处的米灰色麂皮材质增加了质感，显得非常沉稳。
  • 穿搭效果： 这种小面积深蓝色的点缀，既打破了白鞋的单调，又不会像黑白配那样生硬，非常适合 40+ 男士的儒雅气质。

🥈 次选推荐：【AGCV077-1 月白3】（大地棕底款） ​

如果您想要一双更耐看、更有“复古绅士”味道的鞋，选这款。

  • 推荐理由： 它的棕褐色橡胶大底是点睛之笔，这种大地色系与深蓝色裤子是经典的审美组合，视觉上非常协调。
  • 穿搭效果： 棕色底增加了鞋子的“重量感”，能很好地支撑起 180 斤的体型感，不会显得脚下太飘。它比纯白鞋更耐脏，非常适合旅游和日常通勤。

🥉 备选方案：【AGCV077-8 月白3】（纯正藏青色款） ​

这是一款最稳妥、绝不会出错的配色。

  • 推荐理由： 这双鞋直接采用了**藏青色（Navy Blue）**作为主要的装饰色。
  • 穿搭效果： 这种配色方案在视觉上能与您的裤子融为一体，产生视觉延伸感，从而让 178cm 的身高看起来更加修长。它消除了您之前担心的“对比度太高”的问题，整体感极强。

为什么不推荐其他配色？ ​

  • 避开 AGCV077-16 (全灰)： 灰色太重容易显得老气，且配深蓝裤子容易显得灰头土脸。
  • 避开 AGCV077-15 (军绿)： 绿蓝配非常考验搭配技巧，容易显得杂乱。
  • 避开 AGCV077-3 (牛仔蓝)： 鞋面的水洗蓝布料与您的深蓝牛仔裤颜色太接近但又不完全一样，会产生一种“撞色失败”的尴尬感。

购买 ​

  • 实际购买时，才发现推荐第一名的【AGCV077-14 月白3】（云纹蓝红款）几乎没有商家有卖，估计可能是特殊的限量版本。唯一找到的商家开价500多，是普通鞋的2倍多的价格。
  • 转而购买推荐第二名的【AGCV077-1 月白3】（大地棕底款），这个看来是主力款型，很多商家都有卖，价格在200出头一些。

验证 ​

AGCV077-1 月白3 ​

  • 中文名：轻云科技
  • 英文名：Cloud Lite
  • 定位：李宁中端缓震材料，介于基础EVA和高端䨻（Bèng）之间
  • 特点：
    • 比标准“云科技”（Cloud）更轻
    • 回弹响应更快
    • 缓震极限略低，但更适合日常行走与轻运动
关注点Cloud Lite 的实际表现
大体重缓震提供中等缓震，优于普通EVA，适合城市通勤/旅游
是否太软？不会！轻云比标准云更紧实，不易塌陷，稳定性更好
是否适合走路？是！轻量+适度回弹，长时间行走不累脚

对照组 ​

AGCU275-8 月白2 ​

Comfoma ≈ 升级版普通EVA，适合日常走路，但不适合高强度或长时间负重行走。

  • “Comfoma” 是 李宁自主研发的一种基础缓震中底材料，属于其入门级舒适性发泡技术，定位低于“云科技”（Cloud）和“轻云科技”（Cloud Lite）。
  • 材质本质：高回弹EVA（乙烯-醋酸乙烯酯共聚物）的改良配方，比普通EVA更柔软、更轻；
  • 定位：主要用于休闲鞋、板鞋、低强度运动鞋，强调“脚感柔软”而非“高性能缓震”；
  • 性能特点：
    • 初上脚柔软舒适
    • 缓震能力弱于云科技/轻云
    • 长期使用后易压缩塌陷（尤其大体重用户）
    • 无显著回弹反馈，偏“软塌”感

ARZU001-14 ​

  • 李宁 烈骏7 Pro V2 减震回弹专业跑鞋

  • 定位：大体重跑者 / 入门跑者 / 长距离慢跑

  • 核心科技：

    • 中底：全掌李宁云科技 + 后跟稳定框架（Tuff RB）
    • 外底：高耐磨 GCR 橡胶（全覆盖）
    • 鞋面：工程网布 + 热熔膜支撑，包裹性强

  • 特点：

    • 中底厚实（heel stack ≈32mm），缓震强；
    • 后跟加宽+内侧支撑，适合低足弓/扁平足；
    • 重量偏重（约280g/42码），但稳定性极佳；
    • 配色 -14：多为 “曜石黑 / 荧光黄” 或 “深灰蓝 / 白”（部分渠道）

  • 适用场景：日常慢跑、健走、长时间站立（如保安、医护）

✅ 优势：缓震足、支撑稳、耐穿，是李宁跑鞋中少有的“大体重友好”型号。

ARBS001-8 ​

  • 李宁 超轻19 减震透气专业竞速跑鞋
  • 定位：竞速 / 速度训练 / 马拉松
  • 核心科技：
    • 中底：䨻（Bèng）轻弹科技
    • 外底：GCU 超临界发泡橡胶（轻薄、高回弹）
    • 鞋面：䨻丝 mono yarn + 轻量化纱（单层透气）
  • 特点：
    • 极致轻量（≈180g/42码）
    • 中底薄、硬度高，强调能量反馈而非缓震
    • 无足弓支撑、无抗扭片，对脚踝/足弓要求高
    • 配色 -8：常见 “标准白 / 幻影蓝” 或 “电光紫 / 黑”
  • 适用场景：5K–半马速度跑、间歇训练

⚠️ 劣势：缓震弱、支撑差、不耐久，大体重或低足弓用户极易受伤。

ARMS025-5 ​

  • 李宁 赤兔5 Pro 跑步鞋

  • 赤兔5 Pro 是一款专为跑步设计的竞速训练鞋，它在跑鞋中属于“入门稳定型”（Pro 版带 Tuff RB 后跟支撑）。主打轻量、回弹、速度感，适合 5K–半马节奏跑。**

  • 上市时间：2023 年第四季度（2024–2025 年持续销售，2026 年仍在售）

  • 定位：速度训练 / 节奏跑 / 入门竞速跑鞋

  • 它不是缓震跑鞋，不是综训鞋，不是通勤鞋——它的全部设计都围绕“向前跑”展开。如果你主要用途是跑步，它值得考虑。

  • 核心技术参数

项目参数
中底科技全掌䨻（Bèng）轻弹科技 + 后跟 Tuff RB 稳定框架（Pro 版特有）
外底材质GCU 超临界发泡橡胶（轻量、高回弹、抓地强）
鞋面材质工程网布 + 热熔膜支撑条（䨻丝 mono yarn 结构，轻薄透气）
单只重量≈ 210 克（EU 42 码）
中底厚度前掌 ≈ 22mm，后跟 ≈ 28mm
落差（Heel-to-Toe Drop）6mm
适用场景5K–半马节奏跑、间歇训练、日常慢跑
适合脚型正常足弓 / 轻度内旋（低足弓可尝试，但非专为扁平足设计）

主要特点 ​

1. 全掌䨻中底（Bèng） ​

  • 李宁顶级回弹材料，能量反馈率高达 80%+；
  • 比 EVA 或云科技更轻、更弹，适合追求“推进感”的跑者；
  • ⚠️ 缓震偏弱，大体重（>80kg）用户长距离可能感觉冲击明显。

2. Tuff RB 后跟稳定框架（Pro 版专属） ​

  • 在普通赤兔5 基础上增加的硬质 TPU 后跟杯；
  • 提升后跟锁定性，对轻度低足弓或入门跑者更友好；
  • ❌ 但无足弓支撑片或抗扭片，稳定性仍弱于烈骏7等支撑系跑鞋。

3. 轻量化设计 ​

  • 鞋面采用单层工程网布，透气性极佳；
  • 整鞋重量控制在 210g 左右，属于轻量训练鞋范畴。

4. GCU 外底 ​

  • 覆盖前掌和后跟关键区域；
  • 抓地力强（尤其塑胶跑道），但耐磨性一般，不适合粗糙水泥地长期使用。

AGLT131-2 ​

  • 李宁灵动跑鞋（AGLT131），入门级轻量跑鞋
  • 它的“舒适感”来自柔软中底和轻量化设计，但缺乏支撑、稳定性和耐久性，无法满足大体重或长时间行走的生物力学需求。

核心问题 ​

  • 前30分钟，软鞋让你“踩在云上”；30分钟后，你的脚开始“找地”。
  • “软 ≠ 舒适”，“轻 ≠ 省力”

很多人误以为“鞋软就是舒服”，但对大体重用户来说： - 过软的中底 = 缺乏支撑 = 足弓塌陷 + 小腿代偿发力 - 过轻的鞋体 = 缺少结构 = 脚踝不稳定 = 肌肉持续紧张

缺陷分析 ​

1. 中底材质：普通EVA 或 Comfoma（非云科技） ​
  • 商品页仅写“减震”，未标注“云科技”或“轻云” → 极可能为 Comfoma 或基础EVA；
  • 这类材料初上脚柔软，但密度低、易压缩；
  • 大体重下，中底迅速塌陷，失去缓震能力，脚底直接感受地面冲击。
2. 无足弓支撑 & 无抗扭设计 ​
  • 鞋底是一整块软泡棉，没有足弓隆起或TPU抗扭片；
  • 低足弓用户走路时，足弓过度拉伸，引发足底筋膜疲劳；
  • 长时间行走后，小腿、膝盖甚至腰部都会代偿受力。
3. 鞋底过薄 + 外底耐磨差 ​
  • 中底厚度约 24–26mm（后跟），低于通勤鞋标准（28mm+）；
  • 外底橡胶覆盖率低，水泥地行走几周就磨平，进一步加剧疲劳感。
4. 鞋面过软，包裹不足 ​
  • 虽然“网面透气”，但缺乏热熔膜或支撑条；
  • 脚在鞋内轻微滑动，每一步都需要脚趾抓地稳定，累积疲劳。
