## v1.0 版本

- **2025-12-08**  
  **VPCustomContainer组件中图片点击能弹框放大**  
  文章中图片点击后，能弹框放大展示，弹框图片支持鼠标滚轴缩放和鼠标拖拽

- **2025-12-07**  
  **VPCustomContainer组件中章节列表改为树状结构**  
  文章右侧提供章节列表，以树状结构展示，可以收缩展开节点，便于标题多的文章

- **2025-12-06**  
  **VPCustomContainer组件中加入章节列表**  
  文章右侧提供章节列表，悬浮，可收缩展开
  
- **2025-12-05**  
  **采用 ::: blog-post ... ::: 语法 实现自定义组件**  
  解决撰写博客时, 不停报错的问题。
  由于 <BlogPost> 和 </BlogPost> 标签对触发LSP解析错误, 报告"Document has been closed"错误。新方案通过 ::: blog-post ... ::: 这种 markdown 语法替代 Vue 组件标签, 避免LSP解析错误
  
- **2025-11-30**  
  **加入历史时间功能**  
  加入历史事件功能，用户可以查看网站历史事件

- **2025-11-29**  
  **Mermaid 图增加缩放、拖拽、弹框显示等功能**  
  Mermaid 图增加功能：鼠标滚轮缩放、鼠标拖拽、点击弹框放大显示且支持缩放拖拽
