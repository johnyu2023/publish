## 台儿庄路地铁站

+ 9号线台儿庄路地铁站
+ 前面就是平度路，穿过杨高中路，就变成了金藏路
+ 路口有联想的维修站
+ 附近有个自助称重快餐店
+ 附近有极乐汤、啦啦宝都
