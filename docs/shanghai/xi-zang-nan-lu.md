+ 西藏南路连接浦东和浦西的重要道路

## 西藏南路地铁站

+ 8,4号线西藏南路地铁站
+ 新7天酒店住宿，不提供早餐，马路对面是汉庭，如家
+ 路口有聚盒算NB的店
+ 附近有大境中学，大境初级中学
+ 附近有世和酒店，Tims，KFC
+ 附近有九院、江南智造生活广场
+ 稍远处有五爱高中
+ 再远点有卢湾工人体育场（现邮币市场）
+ 再远点有博荟广场
